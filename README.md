# Quantum J Toolkit

A Python-based computational toolkit for exploring angular momentum operators in quantum mechanics.

This project provides matrix, algebraic, and differential representations of quantum angular momentum operators, along with support for Dirac notation and spherical harmonics.

---

## Features

- Matrix representation of angular momentum operators (Jx, Jy, Jz, J+, J-)
- Ladder operator action on quantum states |j, m⟩
- Operator chain evaluation (e.g. J+ J- |j, m⟩)
- Commutator calculations
- Differential operator representation in (θ, φ)
- Spherical harmonics Yₗᵐ(θ, φ)

---

## Requirements

- Python 3.8+
- sympy
- numpy

Install dependencies:

```bash
pip install sympy numpy
```
